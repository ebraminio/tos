If you want to use the service with a language other than Persian add English, add ISO 639-3 code of it at the end of url, something like:

https://tools.wmflabs.org/tesseract-ocr-service/eng or https://tools.wmflabs.org/tesseract-ocr-service/fas+eng (default)
